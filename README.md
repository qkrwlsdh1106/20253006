# Linux 명령어 top, ps, jobs, kill

# 1. top
   * **'top'** 명령어는 리눅스 시스템에서 실행 중인 프로세스와 시스템의 자원 사용 현황을 실시간으로 보여줌

### 기본 사용법
  * 터미널에서 'top' 명령어 입력하고 엔터를 누르면 된다.

### 주요 기능
  * **실시간 모니터링**: 'top' 명령어는 일정 주기로 화면을 업데이트하여 시스템 성능을 실시간으로 모니터링 할 수 있다.
  * **프로세스 정보**: 실행 중인 프로세스의 PID, 사용자, 우선순위, 메모리 사용량, CPU사용량 등을 보여준다.
  * **시스템 요약**: 화면 상단에 시스템의 전반적인 성능 지표를 요약하여 표시한다. 이는 시스템 가동 시간, 사용자 수, 로드 평균, CPU 상태, 메모리 사용량, 스왑 공간 사용량 등을 포함한다.
